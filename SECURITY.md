# Security

## Supported versions

Security fixes are tracked on the current public branch and released as needed.

## Reporting a vulnerability

Do not open a public issue for a security problem.

Send a private report to the maintainers with:

- a clear description of the issue
- affected paths, versions, or environments
- reproduction steps if available
- any suggested mitigation

## What happens next

We will acknowledge the report, investigate, and coordinate disclosure and remediation as appropriate.

If the issue is confirmed, we will work toward a fix before public disclosure.
